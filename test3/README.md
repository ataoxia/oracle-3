#### 给自己的账号分配USERS,USERS02,USERS03三个分区的使用权限
```sql
ALTER USER C##new_user0 QUOTA 50M ON USERS;
ALTER USER C##new_user0 QUOTA 50M ON USERS02;
ALTER USER C##new_user0 QUOTA 50M ON USERS03;
```
